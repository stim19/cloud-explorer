# clouds-file

A simple Flask based file manager with user authentication.

# Setup Instructions
### Clone the repository
```git clone```
### Install dependencies

```pip install -r requirements.txt```

### Run the app
```flask run```

### Access the app:
Open `http://127.0.0.1:5000` in your browser.
